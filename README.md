# alphabet-quiz

A game to learn (or test yourself on) the order of letters in the Latin alphabet.

## Building

```
make
```

## Preview

```
$ ./alphabet-quiz
When you're done playing, press CTRL+C, CTRL+D or type "quit".

Which letter comes first in the alphabet: X or L?
```
